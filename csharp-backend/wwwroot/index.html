🚀 Agilpay Backend - C# ASP.NET Core

Backend equivalente desarrollado en C# para integración con Agilpay

📦 Endpoints de Productos

GET /api/agilpay/products

Obtiene la lista de productos disponibles

💳 Endpoints de Pagos

POST /api/agilpay/create-payment

Crea una solicitud de pago con Agilpay

POST /api/agilpay/payment-response

Maneja la respuesta de Agilpay después del pago

👥 Endpoints de Usuarios

GET /api/users

Obtiene todos los usuarios

POST /api/users

Crea un nuevo usuario

GET /api/users/{id}

Obtiene un usuario específico

PUT /api/users/{id}

Actualiza un usuario existente

DELETE /api/users/{id}

Elimina un usuario

🔧 Tecnologías Utilizadas

  • ASP.NET Core 8.0 - Framework web
  • Entity Framework Core - ORM para base de datos
  • SQLite - Base de datos
  • Swagger/OpenAPI - Documentación de API
  • Newtonsoft.Json - Serialización JSON

🚀 Cómo ejecutar

  1. Asegúrate de tener .NET 8.0 SDK instalado
  2. Navega al directorio del proyecto
  3. Ejecuta: dotnet restore
  4. Ejecuta: dotnet run
  5. Accede a https://localhost:5001 o http://localhost:5000
  6. Visita /swagger para la documentación interactiva de la API
